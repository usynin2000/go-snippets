# 📋 План повторения Go (от простого к сложному)

## **Фаза 1: Восстановление базы (День 1-2)** 🎯

[x] 1. **`theory/base.md`** - начните отсюда, освежите базовый синтаксис
[x] 2. **`hello_world.go`** - запустите, вспомните как работает Go
[x] 3. **`bool/`** - базовые булевы операции (4 файла)
[x] 4. **`cycles/`** - циклы и итерации (7 файлов)
[x] 5. **`stanardlib/`** - базовые функции стандартной библиотеки

## **Фаза 2: Фундаментальные концепции (День 3-4)** 🏗️

[x] 6. **`functions/`** - все о функциях (9 файлов: вариадики, замыкания, рекурсия)
[x] 7. **`pointers/`** - указатели + практика (важно для понимания Go!)
[x] 8. **`struct/`** - структуры данных
[x] 9. **`interfaces/`** - интерфейсы с теорией (критически важно для Go!)

## **Фаза 3: Работа с коллекциями (День 5)** 📦

[x] 10. **`array_and_slices/`** - массивы и слайсы + практика
[x] 11. **`map/`** - мапы (15 файлов)
[x] 12. **`strings/`** - работа со строками

## **Фаза 4: Обработка ошибок (День 6)** ⚠️

[x] 13. **`errors/`** - теория и практика ошибок (6 файлов: panic, recover, кастомные ошибки)

## **Фаза 5: Конкурентность (День 7-8)** 🚀

[+-] 14. **`gorutines_channels/`** - горутины, каналы, worker pools (7 файлов)
[x] 15. **`WaitGroup.go`** - синхронизация
16. **`context/`** - управление контекстом (критично для реальных проектов!)

## **Фаза 6: Веб и HTTP (День 9-10)** 🌐

17. **`net-http/`** - HTTP сервер, роутинг, middleware (11 файлов + chi, resty)
18. **`MiniServer.go`**, **`server.go`** - практические примеры

## **Фаза 7: Продвинутые темы (День 11-12)** 🎓

19. **`generics/`** - дженерики (7 файлов)
20. **`tags/`** - теги структур
21. **`logs/`** - логирование (logrus, zap)
22. **`testing_utils/`** - тестирование

## **Фаза 8: Дополнительно** 📚

23. **`flag/`** - аргументы командной строки
24. **`os/`** - работа с окружением

---

## 🎯 Рекомендуемый подход:

1. **Каждый день выбирайте 1-2 темы** из плана
2. **Читайте `.md` файлы** (теория) перед `.go` файлами
3. **Запускайте примеры**: `go run filename.go`
4. **Модифицируйте код** - меняйте примеры, экспериментируйте
5. **Делайте практику** - во многих папках есть подпапки `practice/`

## ⚡ Экспресс-вариант (если времени мало):

Если нужно быстро восстановить навыки, сфокусируйтесь на:
- `theory/base.md` → `functions/` → `pointers/` → `struct/` → `interfaces/` → `gorutines_channels/` → `context/` → `errors/`

Это даст вам 80% необходимых знаний для работы.
